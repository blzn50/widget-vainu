Awesome Client using Vainu Widget

Load vainu widget whereever you want in your webpage.

Follow the instruction

1. Get the script from vainu

(function (w, d, s, o, f, js, fjs) { w['vainu-widget'] = o; w[o] = w[o] || function () { (w[o].q = w[o].q || []).push(arguments) }; js = d.createElement(s), fjs = d.getElementsByTagName(s)[0]; js.id = o; js.src = f; js.async = 1; fjs.parentNode.insertBefore(js, fjs); }(window, document, 'script', 'vainu', 'script url'));

2. Load the script

vainu('load', "section-right", "12345678");
Required parameters
load - API method to run the script
section-right - Class name of the webpage to load the vainu widget into
12345678 - Business id to fetch data from vainu
